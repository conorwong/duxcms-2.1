#DUXCMS 2.0 开发版

DUXCMS是一款针对于中小网站与二次开发平台的一款管理系统，基于canphp3框架，后台采用拼图UI，更加利于多平台使用于开发者快速开发

开发版基于正式版进行的部分优化于实验性功能，不建议作为正式建站使用

环境支持：
php5.3+
mysql5.1

浏览器支持：
ie8+ chrome firefox 
因特性问题暂不支持IE8以下浏览器，请升级您的浏览器
